# KONEX CREATIVE

Plataforma de geração de currículos profissionais com inteligência artificial.

## Requisitos

- PHP 7.4+
- MySQL / MariaDB
- Extensão cURL habilitada no PHP

## Instalação

1. Clone ou faça o upload dos arquivos para o servidor.
2. Importe o arquivo `database.sql` no seu banco de dados MySQL/MariaDB.
3. Copie `.env.example` para `.env` e preencha as variáveis com os valores reais do seu ambiente.
4. Configure as credenciais de banco de dados e demais variáveis no `api.php` com os valores do seu ambiente.

## Variáveis de Ambiente

Consulte o arquivo `.env.example` para ver todas as variáveis sensíveis que precisam ser configuradas antes de colocar o projeto em produção.
